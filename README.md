# AI Case Processing Service

Microservicio de procesamiento inteligente de casos basado en LLM.

## Stack

- FastAPI
- PostgreSQL
- Groq (llama-3.1-8b-instant)
- Docker

## Quick Start

### Requerimientos

- Docker Desktop instalado y corriendo

### Correr el sistema

Clonar el repositorio:

```bash
git clone https://github.com/your-user/ai-case-processing-service.git
cd ai-case-processing-service
```

Correr el contenedor:

```bash
docker compose up --build
```

http://localhost:8000/docs
